Gujsingh
-----------
DESCRIPTION:
Battleship game played between two u32 boards with Pic32 microcontrollers
-Implements a low-level parser based on the NMEA0183 protocol
-Implementation of hierarchal state machine
-Embedded communication using the UART protocol
-Staff agent with A.I intelligence
-Event Checkers running on IRS
